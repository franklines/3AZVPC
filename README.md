# 3AZVPC
Deploy an AWS VPC with 3 Availability Zones (Public &amp; Private Subnets) using Terraform.

## VPC Diagram

![vpc Diagram](https://i.imgur.com/n3APIiG.png)

## How to deploy

```
[~] export AWS_ACCESS_KEY_ID="<id>"
[~] export AWS_SECRET_ACCESS_KEY="<secret>"
[~] terraform init
[~] terraform plan
[~] terraform apply
```
